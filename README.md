Kurrasch Records Website

A Department of Kurrasch Gaming Merch Store LLC
Location: Minnesota, USA

Overview

The Kurrasch Records Website is the official online platform for Kurrasch Records.
It provides:

Access to music releases and artist information

Integrated dashboards for staff and server owners

Weather and bot-linked dashboards (for Midwest Weather integration)

Announcements and updates directly from the Nexus Staff Panel

Important: This repository contains proprietary code, assets, and designs owned by Kurrasch Records.

Features

Staff Dashboard (Private) — Access for Nexus staff and owner only

Public Dashboard — For verified server owners and general users

Bot Integration — Syncs data and announcements with the Discord bot

Status Page — Live service monitoring similar to DownDetector

Invite Page — Add the Midwest Weather bot to your Discord server

Announcements Page — View system updates, bot news, and weather alerts

Legal Pages — Terms, Privacy Policy, and Copyright info
